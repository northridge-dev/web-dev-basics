Review of HTML Basics

Let's review:

  • HTML stands for Hyper Text Markup Language.
  • At bottom, it's tags that mark up a document in a way that browsers understand.
  • Tags indicate the role or function of some part of the document. For example:
    • <h1> is a heading.
    • <p> is a paragraph.
    • <strong> marks text as important.
  • Tags can have attributes that provide additional information.
  • Attributes are always written in the opening tag.
  • Attributes have a name and a value, separated by an equals sign.
  • Tags can be nested inside other tags. In general, inline tags, like <em> can be nested inside block tags.
  • Some tags can be self-closing.
  • <a> is a very special tag. It's what makes it possible to link together parts of the Web.
