Review of Basic CSS

  • CSS stands for:
    • Cascading
    • Style
    • Sheets
  • CSS is used to style web pages
  • You can apply styles selectively:
    • to a specific element, with an inline style (this bullet point is styled with an inline style)
    • to a specific element by ID (this bullet point is styled with an ID selector)
    • to a group of elements by class (this bullet point is styled with a class selector)
    • to all elements created with a specific tag (this bullet point is styled with a tag selector)
  • The selectors, above, are listed in order of specificity. When there are conflicts between style instructions, the more specific instruction is applied.
  • Style properties are set with key-value pairs: style-property: value
